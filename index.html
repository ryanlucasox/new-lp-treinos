Upper30

Configure suas credenciais do Firebase para ativar login, sincronização entre dispositivos e histórico de treinos na nuvem.

1
Acesse console.firebase.google.com e clique em "Criar projeto". Dê um nome (ex: upper30) e desative o Google Analytics.
2
No painel do projeto, clique no ícone </> Web. Registre o app com qualquer nome. Copie o objeto firebaseConfig que aparece.
3
Vá em Authentication → Começar → Google e ative o provedor Google. Depois vá em Firestore Database → Criar banco → Modo de produção.
4
No Firestore, vá em Regras e substitua por:
rules_version='2'; service cloud.firestore { match /databases/{db}/documents { match /users/{uid}/{document=**} { allow read, write: if request.auth.uid == uid; } } }
5
Cole os valores do firebaseConfig nos campos abaixo e clique em Salvar.
API Key
Auth Domain
Project ID
Storage Bucket
Messaging Sender ID
App ID
